# breno.dev — Portfólio

Portfólio pessoal desenvolvido para apresentar minha trajetória, projetos e stack como desenvolvedor Backend em formação.





## Sobre o projeto

Site construído com o conceito visual **"API Response"** — uma identidade inspirada no universo de APIs e terminais, com tema dark, tipografia monoespaçada para elementos técnicos e acentos vibrantes em ciano, violeta e magenta.

O objetivo é reunir em um só lugar minha apresentação, projetos desenvolvidos, stack técnica e formas de contato, além de disponibilizar meu currículo para download.

## Funcionalidades

- Terminal animado no topo simulando uma resposta de API em JSON
- Menu de navegação fixo com destaque automático da seção visível durante a rolagem
- Efeitos visuais: gradientes animados, brilho ao redor de elementos, cards com leve inclinação 3D no hover
- Seção de projetos com cards detalhados (tecnologias usadas, descrição e link)
- Botão para visualizar/baixar o currículo em PDF
- Formulário de contato que abre o cliente de e-mail já preenchido
- Totalmente responsivo (desktop, tablet e mobile)
- Respeita a preferência de redução de movimento do usuário (`prefers-reduced-motion`)

## Tecnologias utilizadas

- **HTML5** — estrutura semântica
- **CSS3** — variáveis, grid, flexbox, animações e efeitos visuais
- **JavaScript** (Vanilla) — interações, scroll reveal, menu mobile, efeito de digitação

## Como rodar o projeto localmente

```bash
# Clone o repositório
git clone https://github.com/Brenodev4/portifolio-Brenodev.git

# Entre na pasta do projeto
cd portifolio-Brenodev
```

Depois é só abrir o `index.html` no navegador, ou rodar com a extensão **Live Server** do VS Code para ter recarregamento automático.
